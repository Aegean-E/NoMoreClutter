# NoMoreClutter

AI-powered file organizer with local LLM support.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## LLM Setup

This app supports local LLMs via Ollama or LM Studio (OpenAI-compatible API):
- **Ollama**: Run `ollama serve` and use `http://localhost:11434/v1`
- **LM Studio**: Run LM Studio and use `http://localhost:1234/v1`
